# 7569번 : 토마토
### 토마토를 창고에 보관하는 격자모양의 상자들의 크기와 익은 토마토들과 익지 않은 토마토들의 정보가 주어졌을 때, 며칠이 지나면 토마토들이 모두 익는지, 그 최소 일수를 구하는 프로그램 작성하기
#### 조건
- 상자의 가로 칸의 수 ```M```과 세로 칸의 수 ```N``` 그리고 쌓아올려지는 상자의 수 ```H```가 주어짐.
  (2 <= M, N <= 100, 1 <= H <= 100)
- 익은 토마토는 ```1```, 익지 않은 토마토는 ```0```, 그리고 토마토가 들어있지 않은 칸은 ```-1```로 나타냄.
- 익은 토마트는 하루가 지나면 인접한 ```위, 아래, 왼쪽, 오른쪽, 앞, 뒤```에 있는 익지 않은 토마토를 익게함.
- 저장할 때부터 모든 토마토가 익어있으면 ```0```을 출력, 만약 토마토가 모두 익지 못하면 ```-1```을 출력
### 풀이  
문제의 토마토가 들어있는 상자를 표현하기 위해서 **3차원 배열**을 사용해야한다.  
그래서 이 부분에 어려움을 순간적으로 느낄 수도 있지만, 2차원 배열에서의 구현 방법과 크게 다른 것이 없다!  

이 문제는 ```너비 우선 탐색(BFS)```를 이용하면 해결할 수 있다.  
```BFS```는 ```큐```를 이용해서 제일 가까운 노드부터 탐색하는 방법이기 때문에 현재 익은 토마토를 기준으로 주변에 익지 않은 토마토들을 익게하는 과정을 구현하기에 적합한 방법이다. 또한 최초에 주어진 익은 토마토 개수가 여러 개가 될 수 있기 때문에, ```BFS``` 탐색을 시작하기 앞서 큐에 익은 토마토들을 삽입하고 시작하면 우리가 원하는 결과를 얻을 수 있다!  

다만, 모든 ```BFS``` 탐색이 종료한 뒤에도 익지 못한 토마토가 존재하는 경우가 있기 때문에 이를 꼭! 검사한 뒤에 결과를 출력해야한다.
